Задание L0 
Для запуска необходим docker (https://www.docker.com/)

Порядок запуска:
1) В распакованной с директории запустить следующие команды
docker-compose build
docker-compose up
2) Поднимутся три контейнера (более подробно описанно в docker-compose.yml)
В первом контейнере минимально работоспособный образ posgresql
Во втором контейнере бизнес логика на Golang, скомпилированная с помощью последнего доступного образа Golang
В третьем контейнере брокер сообщений NATS, через который приложение получает заказы
